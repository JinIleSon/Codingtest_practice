# BOJ 10799 - 쇠막대기

https://www.acmicpc.net/problem/10799

## 풀이 방식

**스택을 활용한 열린 쇠막대기 개수 관리**

괄호 문자열을 왼쪽부터 순회하면서 `(`는 스택에 넣고, `)`를 만나면 스택에서 하나를 꺼낸다.  
이때 바로 앞 문자가 `(`이면 레이저, `)`이면 쇠막대기의 끝으로 판별한다.

---

## 핵심 아이디어

```
문자열을 왼쪽부터 한 글자씩 확인

1. '(' 이면 스택에 push
2. ')' 이면 스택에서 pop
3. 직전 문자가 '(' 이면 → 레이저
   -> 현재 열려 있는 쇠막대기 개수만큼 count 증가
4. 직전 문자가 ')' 이면 → 쇠막대기 끝
   -> 막대기 조각 1개 증가
```

예시:

```
( ( ) ( ) )
    ↑
   레이저
```

레이저 `()`가 나오면 그 시점에 스택에 남아 있는 쇠막대기 수만큼 조각이 추가된다.

| 상황 | 연산 | 복잡도 |
|------|------|--------|
| `(` 등장 | `stack.append()` | O(1) |
| `)` 등장 (레이저) | `count += len(stack)` | O(1) |
| `)` 등장 (막대기 끝) | `count += 1` | O(1) |

---

## 시간초과 원인 및 개선

- **비효율적인 방식**: 각 쇠막대기를 따로 추적하며 몇 번 잘렸는지 계산
- **개선 방식**: 현재 열려 있는 쇠막대기 수만 관리

스택을 사용하면 각 괄호를 한 번씩만 처리하므로 전체 시간 복잡도는 **O(n)**이다.

---

## 최종 코드

```python
import sys

input = sys.stdin.readline
pipe = input().strip()

stack = []
count = 0

for i in range(len(pipe)):
    if pipe[i] == '(':
        stack.append('(')
    else:
        stack.pop()
        if pipe[i - 1] == '(':
            count += len(stack)
        else:
            count += 1

print(count)
```

---

## 복습 포인트

- `)`를 만났다고 무조건 쇠막대기 끝이 아니다.
- `()` 형태일 때만 레이저다.
- 스택에는 현재 열려 있는 쇠막대기 수를 담는다고 생각하면 된다.
- 개별 쇠막대기를 추적하지 않고, 현재 열린 개수만 관리하는 것이 핵심이다.
