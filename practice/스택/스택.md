## 학습 주제

파이썬으로 스택 문제를 풀면서 문자열 처리, 입력 처리, 리스트 동작 방식을 함께 익혔다.

---

## 알게 된 함수

### `input()`

> **주의**: 입력이 많은 코딩테스트 환경에서 반복 호출 시 시간 초과가 발생할 수 있다.

- `input()`을 반복적으로 단독 사용하면 성능 문제가 생길 수 있다.
- 명령어 개수가 많은 문제에서는 입력 속도가 성능에 직접 영향을 준다.

---

### `sys.stdin.readline`

> **권장**: 빠른 입력이 필요할 때 사용하는 코딩테스트 표준 패턴이다.

```python
import sys
input = sys.stdin.readline
```

- `readline()`은 입력 문자열 끝에 줄바꿈 문자 `\n`을 포함한다.
- 문자열 비교나 가공이 필요한 경우 `strip()`과 함께 사용한다.

```python
S = input().strip()
```

---

### `len()`

문자열 길이를 반복문 없이 바로 구할 수 있다. 불필요한 반복을 줄여 더 효율적이다.

```python
length = len(S)
```

## 알게 된 점

### 스택 구조 (Stack)

| 개념 | 설명 |
|------|------|
| `push` | 값을 스택에 넣는 동작 |
| `pop` | 마지막에 넣은 값을 꺼내는 동작 |
| 순서 | 먼저 들어간 값은 리스트 앞쪽 인덱스에 위치하지만, 꺼낼 때는 항상 마지막 요소부터 |
| 구조 | 후입선출 — **LIFO** (Last In, First Out) |

---

### 리스트와 스택의 연결

파이썬에서는 리스트를 이용해 스택을 구현할 수 있다.  
`append()`로 값을 넣고, `pop()`으로 마지막 값을 꺼내는 방식이 스택 동작과 일치한다.

```python
stack.append(x)  # push — 값 삽입
stack.pop()      # pop  — 마지막 요소 반환 및 제거
```
