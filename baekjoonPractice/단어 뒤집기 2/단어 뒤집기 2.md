## 문제 설명

문자열 S가 주어졌을 때, `<태그>` 내부의 문자열은 그대로 유지하고, 태그 **외부에 있는 단어만** 뒤집어 출력한다.

### 문자열 규칙

- 알파벳 소문자(`a`-`z`), 숫자(`0`-`9`), 공백(` `), 특수 문자(`<`, `>`)로만 이루어져 있다.
- 문자열의 시작과 끝은 공백이 아니다.
- `<`와 `>`는 번갈아 등장하며, `<`가 먼저 등장한다. 두 문자의 개수는 같다.
- 태그는 `<`로 시작하고 `>`로 끝나는 길이 3 이상의 부분 문자열이며, 내부에는 알파벳 소문자와 공백만 있다.
- 단어는 알파벳 소문자와 숫자로 이루어진 부분 문자열이며, 연속하는 두 단어는 공백 하나로 구분된다.
- 태그는 단어가 아니며, 태그와 단어 사이에는 공백이 없다.

### 입력

첫째 줄에 문자열 S가 주어진다. S의 길이는 100,000 이하이다.

### 출력

첫째 줄에 문자열 S의 단어를 뒤집어서 출력한다.

### 입출력 예시

| 입력 | 출력 |
|------|------|
| `baekjoon online judge` | `noojkeab enilno egduj` |
| `<boj>baekjoon online judge` | `<boj>noojkeab enilno egduj` |
| `<per>a</per>` | `<per>a</per>` |
| `<a>abc</a>` | `<a>cba</a>` |

## 풀이 접근

### 핵심 아이디어

문자열을 순회하면서 현재 위치가 **태그 내부인지 외부인지**를 플래그(`count`)로 추적한다.

- **태그 내부** (`<` ~ `>` 사이): 문자를 그대로 출력
- **태그 외부**: 단어를 `deque`에 쌓아두었다가 단어가 끝나는 시점(공백, `<` 등장, 문자열 끝)에 역순으로 출력

### 단어 끝 판별 조건

다음 세 가지 경우에 `orderQue`에 쌓인 문자들을 역순으로 꺼내 출력한다.

1. 현재 문자가 공백(` `)인 경우
2. 다음 문자가 `<`인 경우 (태그 시작 직전)
3. 현재 위치가 문자열의 마지막인 경우

### 주의 사항

- 공백은 `orderQue`에 넣지 않고, 단어를 출력한 뒤 별도로 공백을 출력한다.
- 태그 바로 앞에서 단어가 끝날 때(`다음 문자 == '<'`)와 문자열의 끝에서는 공백을 출력하지 않는다.
- `>` 처리를 `<` 처리보다 먼저 검사해야 태그 종료가 올바르게 동작한다.

---

## 막혔던 부분

### 1. 공백을 `orderQue`에 넣지 않아야 한다

공백을 조건 없이 `orderQue`에 넣으면, 단어를 역순으로 꺼낼 때 **공백까지 함께 뒤집혀** 단어 앞에 공백이 붙는 문제가 발생한다.

```python
# 잘못된 예 - 공백도 큐에 쌓임
orderQue.append(que[i])  # que[i]가 ' '일 때도 append

# 올바른 예 - 공백은 건너뜀
if que[i] != ' ':
    orderQue.append(que[i])
```

공백은 단어 구분자 역할만 하므로, 단어를 역순 출력한 뒤 별도로 `print('', end=' ')`로 처리해야 한다.

### 2. 단어 뒤 공백을 출력하지 않아야 하는 경우

단어가 끝났다고 해서 항상 공백을 출력하면 안 된다. 다음 두 경우에는 공백 출력을 생략해야 한다.

| 경우 | 이유 |
|------|------|
| 다음 문자가 `<`인 경우 | 태그와 단어 사이에는 공백이 없다는 문제 조건 때문 |
| 현재 위치가 문자열의 마지막인 경우 | 문자열 끝에 불필요한 공백이 붙는 것을 방지 |

```python
# 공백 출력 전, 생략 조건을 먼저 확인
if (i < len(que) - 1 and que[i + 1] == '<') or i == len(que) - 1:
    continue  # 공백 출력 생략
print('', end=' ')
```

## 코드

```python
import sys
from collections import deque

input = sys.stdin.readline

S = input().strip()

que = deque(S)
orderQue = deque()

count = 0  # 태그 내부 여부 (1: 내부, 0: 외부)

for i in range(len(que)):
    # '>'를 만나면 태그 종료
    if que[i] == '>':
        count = 0
        print(que[i], end='')
        continue
    # '<'를 만나거나 태그 내부라면 그대로 출력
    if que[i] == '<' or count == 1:
        count = 1
        print(que[i], end='')
        continue

    # 태그 외부: 공백은 큐에 넣지 않음
    if que[i] != ' ':
        orderQue.append(que[i])

    # 단어 끝 조건: 공백, 다음 문자가 '<', 또는 마지막 인덱스
    if i == len(que) - 1 or que[i] == ' ' or (i < len(que) - 1 and que[i + 1] == '<'):
        for j in range(len(orderQue)):
            print(orderQue.pop(), end='')
        # 태그 직전이거나 마지막이면 공백 출력 안 함
        if (i < len(que) - 1 and que[i + 1] == '<') or i == len(que) - 1:
            continue
        print('', end=' ')
```
